For Level 5 to 6, I started by using **cd /krypton/krypton5*. I read the **README** first, then viewed all three ciphertext files using **cat found1, cat found2,** and **cat found3** to see what I was working with. After that, I checked the next‑level file, **krypton6**, to gain additional information. Since the **README **mentioned that no key length would be provided, I copied the text from **found1** and used **Dcode’s Vigenère Automatic Decryption tool**. Dcode handled the analysis for me—it detected both the key length and the key itself. The key turned out to be **KEYLENGTH** and using it to decrypt the challenge ciphertext granted me the key for the next level which is **RANDOM**.
