For Level 2 to 3, I started by navigating to the challenge directory with cd **/krypton** and then into **krypton2** to read the **README** file. The instructions explained that this level uses a **Caesar cipher**, so I needed to figure out the shift. I created a temporary folder with **mktemp -d** and opened the directory, then I made a **symbolic link** to **keyfile.dat** using **ln -s /krypton/krypton2/keyfile.dat** and set permissions with **chmod 777** . Finally, I used **Cryptii** to decode the encrypted key with this shift, which gave me the password for the next level: **CAESARISEASY**.
