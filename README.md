# Krypton Cryptanalysis Lab
Security lab demonstrating cryptography attacks, frequency analysis, and Linux-based problem solving.

---

## Objectives

Challenges Completed
- Level 0–1: Base64 decoding 
- Level 1–2: ROT13
- Level 2–3: Caesar Cipher
- Level 3–4: Frequency Analysis
- Level 4–5: Vigenère Cipher
- Level 5–6: Key-Length Discovery
- Level 6–7: Cipher Pattern Analysis
